Ventio - نسخة إصلاحات جذرية

التعديلات المطبقة:
1) أول تشغيل التطبيق:
   - لم يعد جهاز Client مجبراً على إنشاء مستخدم محلي قبل المزامنة.
   - الترتيب الجديد: إعداد Host/Client أولاً، ثم تسجيل الدخول/إنشاء Admin عند الحاجة.

2) مشكلة تكرار المستخدمين:
   - عند استيراد/مزامنة المستخدمين يتم منع تكرار نفس username.
   - إذا وجد مستخدمان بنفس الاسم، يتم الاحتفاظ بالنسخة الأحدث بدل إنشاء مستخدمين بنفس الاسم وكلمات مرور مختلفة.
   - تسجيل الدخول يبقى رافضاً لأي تضارب أمني بدل اختيار مستخدم عشوائياً.

3) Host / Client و IP:
   - حفظ Host/Client أصبح فورياً من صفحة الإعدادات.
   - الـ IP الذي يدخله المستخدم لا يتم استبداله تلقائياً بـ 0.0.0.0.
   - عند التحويل بين Host و Client يتم تشغيل/إيقاف خدمة المزامنة مباشرة وتخزين الحالة.

4) المزامنة والأداء:
   - تم إضافة تنظيف تلقائي لسجل المزامنة القديم بعد نجاحه لتخفيف التراكم.
   - تقليل الحمل المتكرر الناتج عن سجلات sync القديمة.

5) صفحة Sales على الموبايل:
   - تحسين العرض للشاشات الضيقة.
   - تقليل العناصر العلوية على الشاشات الصغيرة.
   - جعل أزرار كمية المنتج لا تكسر العرض.
   - عرض آخر 50 فاتورة فقط داخل اللوحة السريعة لتخفيف البطء.

6) Dark / Light Mode:
   - إضافة خيار Theme داخل Settings:
     System / Light / Dark
   - يتم حفظ اختيار المستخدم وعدم إجباره على إعدادات الجهاز فقط.
