Ventio Database SQL Editor update

Files changed:
- lib/features/database/database_page.dart
- lib/core/services/database_sql_editor_service.dart

What was added:
1. Added a professional SQL Editor mode to the Database page menu.
2. Added Query Mode for SELECT / WITH / EXPLAIN statements.
3. Added Execute Writes mode for INSERT / UPDATE / DELETE / REPLACE statements.
4. Added confirmation dialog before write SQL statements run.
5. Blocked dangerous schema/system SQL commands:
   - DROP
   - ALTER
   - ATTACH
   - DETACH
   - VACUUM
   - REINDEX
   - PRAGMA writable_schema
   - PRAGMA journal_mode
   - PRAGMA synchronous
6. Added multi-statement script parsing that respects quotes and SQL comments.
7. Added automatic LIMIT 200 for query previews when the query has no LIMIT.
8. Added result table rendering for SELECT output.
9. Added helper actions:
   - List tables
   - SELECT current table
   - Clear
10. After SQL execution, the app refreshes local state through refreshAfterDatabaseChange.
11. SQL changes remain local only and do not create sync changes.

Notes:
- dart/flutter analyze could not be run in this environment because Dart/Flutter is not installed.
- Write SQL is intentionally guarded, but it is still an admin feature and should be used carefully.
